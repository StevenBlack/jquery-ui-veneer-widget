Demo for jquery.ui.veneer.js

A jQuery-UI widget to pannelize vanilla HTML

Author: Steven Black steveb@stevenblack.com Last updated: March 29, 2010.

On GitHub, a branch of jquery-ui: http://github.com/StevenBlack/jquery-ui/tree/veneer/ui/ -- Here's jquery.ui.veneer.js and jquery.ui.veneer.css used on this page.

UI-Veneer converts generic HTML into panels based either on containership or header hierarchy.

 module Foo
		   module Base
		     class Abstract
		       attr_reader :name
		     end
		     def Base.new( name )
		       concrete_class = const_get( name )
		       concrete_class.new
		     end
		     class Bar < Abstract
		       def initialize; @name='--BAR--'; end
		     end
		     class Baz < Abstract
		       def initialize; @name='--BAZ--'; end
		     end
		   end
		 end
		# Try it out...
		puts Foo::Base.new('Bar').name
