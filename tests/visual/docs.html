Overview

Overview:

Problem

You have HTML that you want to decorate and ameliorate with jQuery-ui for viewing in a web browser.

You want to...

  • ...keep your content generic,
  • ...avoid "tag soup",
  • ...benefit from jquery-ui goodness.

Illustrated here

Leverage the standard <h1>, <h2>, <h3>... tags that already exist in your content as markers with which to infer implicit, hierarchical containership.

Options

  • disabled

    Type:
    Boolean
    Default:
    false

    Disables (true) or enables (false) the veneer interactions. Can be set when initialising (first creating) the veneer, or thereafter by changing the option.

    Code examples

    Initialize a veneer with the disabled option specified. 
    $( ".selector" ).veneer( { disabled: true } );
    Get or set the disabled option, after init. 
    //getter
var disabled = $( ".selector" ).veneer( "option", "disabled" );
//setter
$( ".selector" ).veneer( "option", "disabled", true );

  • icons

    Type:
    Object
    Default:
    { 'header': 'ui-icon-triangle-1-e', 'headerSelected': 'ui-icon-triangle-1-s' }

    Icons to use for headers. Icons may be specified for 'header' and 'headerSelected', and we recommend using the icons native to the jQuery UI CSS Framework manipulated by jQuery UI ThemeRoller

    Code examples

    Initialize a with the icons option specified. 
    $( ".selector" ).( { icons: { 'collapsed': 'ui-icon-plus', 'open': 'ui-icon-minus' } } );
    Get or set the icons option, after init. 
    //getter
var icons = $( ".selector" ).( "option", "icons" );
//setter
$( ".selector" ).( "option", "icons", { 'header': 'ui-icon-plus', 'headerSelected': 'ui-icon-minus' } );

Events

Methods

  • destroy

    Signature:
    .( "destroy" )

    Remove the functionality completely. This will return the element back to its pre-init state.

  • disable

    Signature:
    .( "disable" )

    Disable the veneer

  • enable

    Signature:
    .( "enable" )

    Enable the veneer

  • option

    Signature:
    .( "option" , optionName , [ value ] )

    Get or set any option. If no value is specified, will act as a getter.

  • widget

    Signature:
    .( "widget" )

    Returns the .ui-veneer element.
